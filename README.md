# Workspace Integrations

SDKs for creating [Webex Workspace Integrations](https://developer.webex.com/docs/workspace-integrations).

![Poster](./poster.png)

The workspace integrations is a cloud based framework that lets you connect to the Cisco devices, receive sensor data, invoke commands, read status values and update configs in a secure and controlled manner, without installing anything on the devices.

## SDKs

The SDKs are provided for the following platforms:

* [Workspace Integrations Java](https://github.com/cisco-ce/workspace-integrations-java)
* [Workspace Integrations NodeJS](https://github.com/cisco-ce/workspace-integrations-nodejs)
